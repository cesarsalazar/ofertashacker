No, nadie se quiere hacer rico a partir de este sitio.

Es enserio, este sitio no tiene fines de lucro, por el momento es gratis para todos, y si llegamos a cobrar en algún punto será con el firme y único objetivo de mantener vivo este espacio.

¿Por qué?

El objetivo de Ofertas Hacker es promover el desarrollo de software en países latinoamericanos y encontrar las mejores ofertas de trabajo para los mejores desarrolladores.
